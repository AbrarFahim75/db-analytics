# SQL Queries

This folder contains examples of **SQL query operations** used to retrieve and analyze data from databases.

## Topics Covered

- SELECT statements
- WHERE filtering
- ORDER BY
- GROUP BY
- HAVING
- JOIN operations
- Subqueries
- Window functions

## Example

```sql
SELECT name, city
FROM customers
WHERE city = 'Hamburg'
ORDER BY name;
