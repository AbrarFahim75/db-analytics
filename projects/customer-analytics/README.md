# Customer Analytics

This project demonstrates how SQL can be used to analyze customer and order data.

## Objectives

The queries in this project answer common business questions such as:

- How many customers exist in each city?
- Which customers spend the most money?
- What is the average order value?

## Tables Used

- customers
- orders

## Example Queries

The analysis includes queries for:

- customer distribution by city
- customer spending analysis
- average order values
