# DB Analytics

SQL, database management, and analytics projects including schema design, complex queries, and performance optimization.

## Repository Structure

datasets/ – sample datasets used for SQL analysis  
docs/ – database theory and notes  
schemas/ – database schema definitions and ER diagrams  

sql/ – SQL scripts organized by topic

- ddl – Data Definition Language
- dml – Data Manipulation Language
- queries – core SQL queries
- joins – join operations
- subqueries – nested queries
- window-functions – analytical SQL functions
- optimization – performance and indexing

## Topics Covered

- SQL fundamentals
- Joins
- Aggregations
- Subqueries
- Window functions
- Query optimization
